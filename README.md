# 211cc
A small C compiler, named 211cc. This compiler generated x86_64 code as stack machine emulated.

## Build

Run make to build:

```
$ make
```

211cc comes with tests. To run the tests, give "test" as an argument:

```
$ make test
```

For self hosting test below:

```
$ make test-stage2
```
